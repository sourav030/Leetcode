<h2><a href="https://www.geeksforgeeks.org/problems/subsets-1613027340/1">Subsets</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">Given an array <strong>arr[]</strong> of distinct positive integers, your task is to find all its subsets.<br></span></p>
<p><span style="font-size: 14pt;"><strong>Note: </strong>You can return the subsets in any order, the driver code will print them in sorted order.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> arr[] = [1, 2, 3]
<strong>Output: </strong>[[], [1], [1, 2], [1, 2, 3], [1, 3], [2], [2, 3], [3]]
<strong>Explanation: <br></strong>The subsets of [1, 2, 3] in lexicographical order are:
[], [1], [1, 2], [1, 2, 3], [1, 3], [2], [2, 3], [3]</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> arr[] = [2, 4]
<strong>Output: </strong>[[], [2], [2, 4], [4]]
<strong>Explanation:</strong>
The subsets of [2, 4] in lexicographical order are:
[], [2], [2, 4], [4]</span></pre>
<div><span style="font-size: 14pt;"><strong>Constraints :</strong></span></div>
<div><span style="font-size: 14pt;">1 ≤ arr.size() ≤ 10</span></div>
